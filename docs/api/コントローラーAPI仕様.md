# コントローラーAPI仕様

## CorrectionController
- `create_request(data)` - 訂正依頼作成
- `update_request(id, data)` - 更新
- `delete_request(id)` - 削除

## LogController
- `log_operation(...)` - ログ記録

## AuthController
- `verify_password(password)` - 認証
